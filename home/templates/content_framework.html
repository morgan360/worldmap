Key Goals

1. Provide a data structure, or blueprint, for how worlds are defined

2. Allow users to build any kind of (fictional, history) worlds using this blueprint

3. Make these world files transferable across tools, games, devices and web

Philosophy

1. The data structure is open source, free to use, and community-defined

2. Users keep full ownership and control over their world data and any publishing choices

3. Allow other developers to build tools on the framework, with any pricing model

4. Offer core OnlyWorlds applications (one mobile, one desktop, one web) for free and fully functional

5. Promote collaborative development on the framework, its tools, and worlds themselves
