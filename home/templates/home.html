Welcome to OnlyWorlds

A World Standard

OnlyWorlds is a proposal for how we structure our worlds: an open source data framework that defines worlds as interchangeable files, usable in any tool that is made compatible with the format. OnlyWorlds is currently in closed alpha. See the About page for more information and to request access to the initial toolset.
