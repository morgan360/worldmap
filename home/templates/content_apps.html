Flutter project developed in 2023 and 2024 as a mobile world creation interface that helps introduce the framework with some gamification and novel features

{% endcomment %}

Lets you define a world, create and configure elements in it, follow a progression system with some guides and rewards, and chat with AI agents that are specialized for world build assistance.

Installation:

  • iOs app is in TestFlight, a staging environment that you can install from the app store first by following the link
  • Android Google Play is in closed testing. Gain access by joining this Google Group, and then open the link from mobile
  • Android manual install: Download the APK file, transfer to your phone (e.g. to Downloads folder), open through a file viewer, tap the APK file to install

Known bugs and limitations:

  • Number fields only hold positive integer values at this time
  • Time settings are limited, and no such settings exist for weight, height, other units
  • Golden fields do not properly split linked elements
  • Elements with duplicate names cause issues when being linked
  • This being my first Flutter project and mobile app

View and edit elements through a Web Interface.

Or use this Desktop executable (windows).

Build and export your world in the mobile app, then import in Explorer.

Unity project developed in 2023 as an overall world engine, now being reworked to support current framework. Requires a Unity install, but no Unity knowledge.

Files and setup guide on GitHub.

Four sections present a range of world controls:

  • Mapper - For map management, element placement
  • Finder - For browsing worlds
  • Editor - For viewing, editing elements
  • Timeliner - Using a graph node system

Includes map functionality:

  • Maps can be added, saved and configured
  • Pins can be placed, moved, managed, and linked to elements
  • Map data is integrated to the world and exportable
