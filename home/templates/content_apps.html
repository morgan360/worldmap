Flutter project developed in 2023 and 2024 as a gamified mobile world creation interface

Lets you define a world, create and configure elements in it, follow a progression system with some guides and rewards, and chat with AI agents that are specialized for world build assistance

Currently in internal testing environments for both iOs and android

All feedback is welcome: on stability, bugs, ui, content, database, ..

Unity project developed in early 2023 as an overall world 'engine'

Repo and setup guide to be provided shortly (early May 2024)

Wrapper that replaces the entire Unity default editor, prior Unity experience is irrelevant

Four sections present a range of world controls:

  • Finder (browsing the world)
  • Editor (viewing, editing elements)
  • Timeliner (graph node system)
  • Mapper (map management, element placement)

Status per April 2024:

  • Finder is functional
  • Editor is limited to core fields
  • Timeliner has just started development
  • Mapper is functional

Now called Mapper for its most useful section:

  • Maps can be added, saved and configured
  • Pins can be placed, moved, managed, and linked to elements
  • Map data is integrated to the world and exportable

Unity runtime project for world (over)viewing that started development in 2024

Release in early summer

Needs several more weeks of production for:

  • A web-based interface that can load directly from world keys
  • Dynamically creates diagrams for element relations, ownership and assignment

Priority:

  • Desktop / web-based element editing interfaces (including on onlyworlds.com)
  • Mapping tools and a robust mapping data system (see Mapper for an early start)
  • Display tools for world viewing, article and page customization, network graphs, etc
